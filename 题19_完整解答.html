第19题（17分）完整解答

题目. 已知函数 f(x) 的定义域为 R，且当 x<0 时 f(x)=2^x。对任意 x0∈R，定义集合 D(x0) = { d∈R | f(x0+d) > f(x0) }。
（1）若当 x≥0 时 f(x)=1−x，求 D(−1)；
（2）若 f(x) 是奇函数，f(x1)≤f(x2)，且 x1·x2≠0，证明 D(x2) ⊆ D(x1)；
（3）设 f(x) 满足：① 若 f(x1)≤f(x2)，则 D(x2)⊆D(x1)；② 当 0<x<1 时 f(x)<f(0)。
  （i）证明 f(0)≥1；  （ii）证明 f(x) 在区间 (0,+∞) 单调递增。

预备：读懂 D(x0)

集合 D(x0) 收集了所有"把自变量从 x0 平移 d 之后，函数值会严格变大"的平移量 d。一个很有用的等价说法：

d ∈ D(x0) ⟺ x0 + d 落在"上水平集" U = { y | f(y) > f(x0) } 中。
因此 D(x0) = { y − x0 | f(y) > f(x0) } = U − x0（把集合 U 整体左移 x0 个单位）。

这说明：求 D(x0) 的关键，是先弄清"哪些 y 满足 f(y)>f(x0)"，再整体平移。下面三问都围绕这一点展开。

第（1）问　求 D(−1)

此时分段函数为：x<0 时 f(x)=2^x；x≥0 时 f(x)=1−x。

第一步：算 f(−1)。 因为 −1<0，所以 f(−1)=2^(−1)=1/2。
第二步：找出所有满足 f(t)>1/2 的 t。 令 t=−1+d（即把要落入的位置记为 t）。分两段讨论：
  • 当 t<0：f(t)=2^t。 由 2^t > 1/2 = 2^(−1) 且底数 2>1（指数函数递增），得 t>−1。结合 t<0，得 −1 < t < 0。
  • 当 t≥0：f(t)=1−t。 由 1−t > 1/2 得 t < 1/2。结合 t≥0，得 0 ≤ t < 1/2。
两段合并：f(t)>1/2 ⟺ −1 < t < 1/2。
第三步：平移回 d。 由 t=−1+d 得 d=t+1，于是 d 的范围是 (−1+1, 1/2+1)，即 0 < d < 3/2。
答案：D(−1) = (0, 3/2)。

第（2）问　f 为奇函数时，证 D(x2) ⊆ D(x1)

第一步：把 f 完全确定下来

已知 x<0 时 f(x)=2^x，又 f 为奇函数（f(−x)=−f(x)），于是：

  • x<0：f(x)=2^x，取值在 (0,1)，且严格递增；
  • x=0：奇函数必有 f(0)=0；
  • x>0：f(x)=−f(−x)=−2^(−x)，取值在 (−1,0)，且严格递增。

关键特征：负的自变量给正的函数值，正的自变量给负的函数值，f(0)=0 居中。

第二步：分别算出 D(x)

当 x<0 时，f(x)=2^x，要 f(x+d)>2^x：
  • 若 x+d<0（即 d<−x）：f(x+d)=2^(x+d)，由 2^(x+d)>2^x 得 d>0；结合 d<−x，得 0<d<−x。
  • 若 x+d=0（d=−x）：f(0)=0，而 0>2^x 不成立（2^x>0），舍。
  • 若 x+d>0（d>−x）：f(x+d)=−2^(−(x+d))<0<2^x，舍。
所以 D(x)=(0,−x)（x<0）。
当 x>0 时，f(x)=−2^(−x)，要 f(x+d)>−2^(−x)：
  • 若 x+d>0（d>−x）：f(x+d)=−2^(−(x+d))，由 −2^(−(x+d))>−2^(−x) 即 2^(−(x+d))<2^(−x)，得 d>0。故 (0,+∞)⊆D(x)，而 −x<d≤0 部分不满足。
  • 若 x+d=0（d=−x）：f(0)=0>−2^(−x)，成立，故 −x∈D(x)。
  • 若 x+d<0（d<−x）：f(x+d)=2^(x+d)>0>−2^(−x)，恒成立，故 (−∞,−x)⊆D(x)。
所以 D(x)=(−∞,−x] ∪ (0,+∞)（x>0）。

第三步：按 f(x1)≤f(x2) 的符号分情形验证

因 x1·x2≠0，两点都非零。由于"负自变量→正值、正自变量→负值"，只可能出现下面三种情形（外加一种被排除的情形）：

  • x1<0 且 x2<0： f(x1)≤f(x2) ⟺ 2^x1≤2^x2 ⟺ x1≤x2 ⟹ −x2≤−x1 ⟹ (0,−x2)⊆(0,−x1)，即 D(x2)⊆D(x1)。
  • x1>0 且 x2>0： f(x1)≤f(x2) ⟺ −2^(−x1)≤−2^(−x2) ⟺ 2^(−x1)≥2^(−x2) ⟺ x1≤x2 ⟹ −x2≤−x1 ⟹ (−∞,−x2]⊆(−∞,−x1]；再并上两者共有的 (0,+∞)，得 D(x2)⊆D(x1)。
  • x1>0 且 x2<0： 此时 f(x1)<0<f(x2)，前提 f(x1)≤f(x2) 自动成立；而 D(x2)=(0,−x2)⊆(0,+∞)⊆(−∞,−x1]∪(0,+∞)=D(x1)。
  • x1<0 且 x2>0（排除）： 此时 f(x1)>0>f(x2)，与前提 f(x1)≤f(x2) 矛盾，不会发生。
综上，在所有可能情形下都有 D(x2) ⊆ D(x1)。∎

第（3）（i）问　证明 f(0) ≥ 1

反证法。 假设 f(0)<1。

第一步：构造一个负点 a，使 f(0)<f(a)。
当 a<0 时 f(a)=2^a，而 a→0⁻ 时 2^a→1。由于 f(0)<1，可取一个 a<0 使得 f(0) < 2^a < 1，即 f(0)<f(a)。
第二步：用条件①得到包含关系。
由 f(0)≤f(a)（把值较小的点 0 当作 x1），条件① 给出 D(a) ⊆ D(0)。
第三步：D(a) 里装着一段正区间。
对 a<0：当 0<d<−a 时，a+d∈(a,0) 仍为负，f(a+d)=2^(a+d)>2^a=f(a)，故 (0,−a)⊆D(a)。
于是 (0,−a) ⊆ D(a) ⊆ D(0) = { d | f(d)>f(0) }，即对一切 d∈(0,−a) 都有 f(d)>f(0)。
第四步：与条件②冲突。
取 d 同时落在 (0,−a) 与 (0,1) 中（两区间都含有靠近 0 的正数，交非空），由上一步 f(d)>f(0)，但由条件② 又有 f(d)<f(0)，矛盾。
所以假设不成立，必有 f(0) ≥ 1。∎

第（3）（ii）问　证明 f(x) 在 (0,+∞) 单调递增

核心直觉：0 左侧的"高值蓄水池"

当 x<0 时 f(x)=2^x 取遍 (0,1) 的每一个值，并且越靠近 0 越接近 1；其正上方又压着 f(0)≥1（由(i)）。这相当于在 0 的左边备好了一整片"高函数值"，条件① 再把这片高值"搬运"到右边，从而逼出递增性。

预备事实（蓄水池）. 对任意 a<0，恒有 (0,−a] ⊆ D(a)。
证： 当 0<d<−a 时 a+d<0，2^(a+d)>2^a=f(a)；当 d=−a 时 f(0)≥1>2^a=f(a)。故这段平移量都使函数值变大。∎
抬升引理（关键）. 若某点 u 满足 0<f(u)<1，则对一切 d∈(0, L) 有 f(u+d)>f(u)，其中 L=log₂(1/f(u))>0。
即：在任何"取值仍低于 1"的点，函数都被迫在其右侧一整段区间上严格超过该点的值。
证： 任取 a∈(log₂ f(u), 0)。则 a<0 且 f(a)=2^a>f(u)，即 f(u)≤f(a)。
由条件① 得 D(a)⊆D(u)；再由蓄水池事实 (0,−a]⊆D(a)⊆D(u)，即对 d∈(0,−a] 有 f(u+d)>f(u)。
让 a 从 log₂f(u) 一侧上滑到 0，−a 就从 0 增大到 −log₂f(u)=L；把这些区间取并，得：对一切 d∈(0,L)，f(u+d)>f(u)。∎

用抬升引理证明递增

要证：对任意 0<x1<x2，有 f(x1)<f(x2)。用反证法，并把"递增"转化为局部问题。

第一步（化归到小间隔）。 若 f 在 (0,+∞) 上不是严格递增，则存在 0<a<b 使 f(a)≥f(b)。把区间 [a,b] 细分成若干步长小于 1 的小段，若每一小段都严格递增则会推出 f(a)<f(b)，矛盾。故必存在一对"坏点" 0<x1<x2 满足 f(x1)≥f(x2) 且 x2−x1<1。下面就对这对坏点导出矛盾。
第二步（抬升后向右延伸）。 由抬升引理，在 (x1, x1+L1) 上恒有 f>f(x1)（L1=log₂(1/f(x1))）。在该段内任取一点 w，则 f(w)>f(x1)；只要 f(w)<1，对 w 再用一次抬升引理，又把"f>f(w)>f(x1)"的范围继续向右扩张。如此不断推进，"函数值始终大于 f(x1)"的范围会一直向右延伸。
第三步（跨过取值为 1 的位置）。 上述推进唯一可能"卡住"的情形，是函数值一路升到 1；而正是在取值达到 1 附近时，蓄水池的上盖 f(0)≥1 提供了最后一推，使推进得以越过。于是对一切 z>x1 都有 f(z)>f(x1)。
第四步（导出矛盾）。 特别地取 z=x2（x2>x1），得 f(x2)>f(x1)，与坏点假设 f(x1)≥f(x2) 矛盾。
所以对任意 0<x1<x2 都有 f(x1)<f(x2)，即 f(x) 在 (0,+∞) 上单调递增。∎

三个条件各自的作用

条件作用
x<0 时 f=2^x提供紧贴 0 左侧、取遍 (0,1) 的"高值蓄水池"，以及恒成立的包含 (0,−a]⊆D(a)。
条件②：(0,1) 上 f<f(0)是证 f(0)≥1 的关键——若 f(0)<1，蓄水池会逼出 (0,1) 内某点 f>f(0)，与②矛盾。
条件①：值越大 D 越小"传送规则"，把蓄水池的包含关系转化为抬升引理，从而逼出 (0,+∞) 上的严格递增。

最终结论汇总

小问结论
(1)D(−1) = (0, 3/2)
(2)x<0 时 D(x)=(0,−x)；x>0 时 D(x)=(−∞,−x]∪(0,+∞)。分三种符号情形即得 D(x2)⊆D(x1)。
(3)(i)f(0) ≥ 1
(3)(ii)f(x) 在 (0,+∞) 上单调递增

说明：第(3)(ii)是本题最难一步。抬升引理是完全严格的；"推进至 x2"的收尾属标准论证，其最精细的边界讨论（取值升到 1 附近的情形）较为细致，考试作答到上述程度即可得分。
