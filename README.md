## 第19题（17分）

已知函数 f(x) 定义域为 R，当 x<0 时 f(x)=2^x；对任意 x0∈R，定义集合
D(x0) = { d∈R | f(x0+d) > f(x0) }。包含三小问：

- （1）当 x≥0 时 f(x)=1−x，求 D(−1)
- （2）f 为奇函数时，证明 D(x2) ⊆ D(x1)
- （3）在条件 ①②下，证明 f(0)≥1，并证明 f 在 (0,+∞) 单调递增

### 结论速览

- （1）D(−1) = (0, 3/2)
- （2）x<0 时 D(x)=(0,−x)，x>0 时 D(x)=(−∞,−x]∪(0,+∞)，分三种符号情形即得 D(x2)⊆D(x1)
- （3）(i) f(0) ≥ 1；(ii) f 在 (0,+∞) 单调递增

### 解题要点

第 (3) 问的核心是"0 左侧的蓄水池"：x<0 时 2^x 取遍 (0,1) 的所有值，其上方又压着 f(0)≥1。

- (i) 反证：若 f(0)<1，蓄水池会逼出 (0,1) 内某点 f>f(0)，与条件②矛盾。
- (ii) 抬升引理：若 0<f(u)<1，则 f 在 (u, u+log₂(1/f(u))) 上严格大于 f(u)；再借助 f(0)≥1 把"递增"传播到整个正半轴。
